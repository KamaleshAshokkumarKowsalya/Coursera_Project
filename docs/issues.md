# Issue Ready User Stories

## Admin

- As an admin, I want to add and manage doctors so that clinic staffing stays accurate.
- As an admin, I want to manage patient records so that clinic operations stay organized.
- As an admin, I want to view appointment volume so that I can monitor clinic activity.

## Doctor

- As a doctor, I want to view my patient appointments so that I can prepare for visits.
- As a doctor, I want to review patient records so that I can provide better care.
- As a doctor, I want to create prescriptions so that medication guidance is documented.

## Patient

- As a patient, I want to search for doctors by name or specialty so that I can find the right provider.
- As a patient, I want to book and update appointments so that I can manage my schedule.
- As a patient, I want to view prescriptions so that I can follow treatment instructions.
